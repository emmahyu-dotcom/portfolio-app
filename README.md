# React Portfolio Website
This is my personal portfolio website built using React.js. It showcases my projects, skills, and contact information in a responsive and interactive design.

## Live Demo
[https://myportfolio-app-react.netlify.app/](https://myportfolio-app-react.netlify.app/)

## Features
- Home page with introduction and navigation
- About page with bio, education, skills, and interests
- Projects page showcasing multiple projects with links and images
- Contact page with form validation and contact details
- Responsive design for desktop and mobile
- Navigation bar and footer across all pages
- Page transition animations using Framer Motion

## Technologies Used
- React.js
- React Router DOM
- Framer Motion
- HTML5 & CSS3
- JavaScript (ES6+)

- /pages
- /assets
